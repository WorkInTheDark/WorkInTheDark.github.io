# WorkInTheDark.github.io

Personal website for Bingsheng "Arthur" Yao — [workinthedark.github.io](https://WorkInTheDark.github.io).

Built on [Jekyll](https://jekyllrb.com/) with the [al-folio](https://github.com/alshedivat/al-folio) academic theme.

## Local development

Requires Docker Desktop.

```
docker compose up
```

Site serves at http://localhost:8080 with live reload. Edit markdown/SCSS/HTML and the browser refreshes automatically.

To stop:

```
docker compose down
```

## Deployment

GitHub Pages builds from `master` automatically.
